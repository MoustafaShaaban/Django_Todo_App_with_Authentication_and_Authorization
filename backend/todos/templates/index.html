Hello, world!

This is a simple todo app built using Django the awesome web framework.

This project has many functionalities like Create, Read, Update and Delete (CRUD) and user authentication and authorization, and many more!

Built using Bootstrap version 4.6.2 components (Navbar, Jumbotron, Inline-forms, List-groups, etc.) To get started first login to your account by clicking here or register a new account by clicking here. Or just use the navbar! Once logged in, you can use the website or the REST API to list, create, update or delete a todo.
